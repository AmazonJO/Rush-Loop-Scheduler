Fast-track interview scheduling when every hour counts

How to Use This Tool

Generate Template Tab

  1. Enter the candidate's name and time zone
  2. Pick the dates the candidate is available — hour buttons appear automatically
  3. Click the hours the candidate is free
  4. Click "Download Excel Template" — this creates the availability grid
  5. Upload the Excel to SharePoint and share the link with your interviewers
  6. Interviewers type their name (or alias) next to the time slots they're available. Add * for preferred times (e.g. "parinpo*")

Match Schedule Tab

  1. Download the filled-in Excel from SharePoint and upload it here
  2. Review the availability data — set each interviewer's role (Interviewer, Hiring Manager, or Bar Raiser)
  3. (Optional) If you have the Hire Loop Scraper Tampermonkey script, paste the loop JSON to auto-fill roles and competencies
  4. Assign 3 competencies per interviewer using the dropdowns
  5. Click "Find Best Schedule" — the tool matches overlapping availability, prioritizing interviewers with the least options first
  6. Copy the final schedule to clipboard and share

Preferred times (*) are scheduled first. The tool can be re-run as more interviewers submit availability.
